## Decisions Made

- Auto-generate golden files on first run (os.IsNotExist check) so contributors don't need `-update` on initial setup
- Dual golden format per test case: `.txt` captures visual ANSI output, `.json` captures semantic style transitions — enables both layout and color regression detection
- Helpers defined locally in `messages_test.go` rather than a shared package — simpler dependency graph for a test-only pattern
