## Decisions Made

- **helpGoldenPath without width parameter**: The variant name already encodes width (e.g., `3actions-30`), so the path helper takes only `(variant, ext string)` instead of the generic 4-argument `goldenPath`. This keeps filenames consistent with the decision modal golden files pattern.
- **Negative scroll sentinel**: Using `scroll=-1` as a sentinel in `make15()` helper means "compute actual maxScroll" — this avoids hardcoding `6` in tests and makes the fixture self-documenting.
- **checkOrUpdateHelpGolden in same file**: Per-plan helper defined locally in `help_test.go`. Since `*update` flag is declared in `messages_test.go` in the same `package tui`, it's accessible without re-declaration.
