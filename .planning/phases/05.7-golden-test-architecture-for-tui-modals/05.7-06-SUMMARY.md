## Decisions Made
- `MarshalJSON` handles nil Style defensively (adds nil check before json.Marshal) so it works correctly even for `StyleTransition` values constructed with nil Style outside of `ParseANSIStyle`
- `styleMapToArray` initialized with `active := []string{}` literal — ensures non-nil even when no styles active, eliminating the nil/null JSON ambiguity at the source
- Dedup uses replace-last strategy: `result[len(result)-1] = transition` — constant time, no extra data structures needed for the small result sets typical in ANSI output
