## Decisions Made

1. **Manual border construction** — `View()` builds the top border as:
   `"╭── " + styledTitle + fill_dashes + " ──╮"` (char-by-char, lipgloss v2 has no BorderTitle)

2. **wrapBody() stdlib-only** — Uses `strings.Fields` + rune counting; no external wrapping dep to add.

3. **Plain-text test assertions** — Tests assert on View() string content only; ANSI escape code assertions are fragile and platform-dependent (deferred to Phase 06+ visual integration).

4. **Execute-then-feed pattern in integration test** — `_, cmd := m.Update(msg); m.Update(cmd())` because Bubble Tea cmds are lazy functions; modal pop only happens when the cmd is executed and its message fed back.
