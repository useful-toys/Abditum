## What is Phase 05.6?

Fix and complete the **DecisionDialog** — the core modal type for confirmations, acknowledgments and error notifications — which was partially scaffolded in Phase 05.5 but has two compilation errors that prevent the entire `tui` package from building.

## Problem Statement

`internal/tui/decision.go` was added to the working tree but:
1. Has an unused `titleStyle` variable (compile error)
2. Calls `borderStyle.BorderTitle()` — a method that does not exist in lipgloss v2 (compile error)

The package fails to build. All existing TUI tests are blocked.

## Key Deliverables

✅ **Fixed decision.go** — compiles clean, View() builds the title border manually  
✅ **word-wrap helper** — `wrapBody()` for long dialog messages  
✅ **decision_test.go** — 15 PoC fixture tests (all 5 severities × 3 action counts: 1/2/3), each a realistic vault scenario  
✅ **Interaction tests** — Enter/Esc/arrow/unknown-key handling  
✅ **Edge case tests** — long body, short body, Acknowledge no-Esc, small size  
✅ **Integration test** — modal stack push/pop with Enter dispatch
