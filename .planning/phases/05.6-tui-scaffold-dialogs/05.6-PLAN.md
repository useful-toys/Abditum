## Plan 01: Fix decision.go Compilation Errors

**Objective:** Restore the package to a compilable state by fixing the two known errors in `decision.go` and rewriting `View()` with a manual top-border approach (lipgloss v2 has no `BorderTitle`).

**Effort:** ~30 min

### Tasks

1. **Remove unused `titleStyle` variable** (line 179) — error 1.

2. **Replace `borderStyle.BorderTitle(...)` call** with manual border construction — error 2.
   Rewrite `View()` to:
   - Build top border line manually: `╭── {symbol}  {title} ──...──╮`
   - Build body lines: `│  ` + wrapped_body_line + padding + ` │`
   - Keep blank padding lines above/below body (1 line each)
   - Assemble: `[topBorder, emptyLine, bodyLines..., emptyLine, actionBar]`

3. **Add `wrapBody(text string, maxW int) []string` helper** — word-wraps body text to maxW runes per line. Splits on spaces and existing `\n` newlines. Uses only stdlib.

4. **Simplify `View()`** — remove the `rendered = borderStyle...Render(inner)` + line-replacement dance; build the box line by line instead. This is simpler and avoids the lipgloss v2 incompatibility.

### Rendering Detail

**Top border line:**
```
╭── ⚠  Excluir segredo ───────────────────────╮
```

Formula:
```
leftAnchor  = "╭── "                    (4 runes)
rightAnchor = " ──╮"                    (4 runes)  
titleStr    = symbol + "  " + title     (if symbol != "")
            | title                     (if symbol == "")
fillW       = boxW - len(leftAnchor) - len(rune[]titleStr) - len(rightAnchor)
             (min 1)
topLine     = leftAnchor + styledTitle + dashes(fillW) + rightAnchor
```

**Side lines (body content):**
```
│  line_content_padded_to_(boxW-4)  │
```

Formula per body line:
```
inner = "│  " + padRight(line, boxW-4) + " │"
```

**Empty padding lines:**
```
│                                        │
```

**Bottom border:** already built by `renderActionBar()` — unchanged.

**Full assembly:**
```go
lines := []string{topLine, emptyPad}
for _, bl := range wrappedBody {
    lines = append(lines, "│  "+padRight(bl, boxW-4)+" │")
}
lines = append(lines, emptyPad, actionBar)
return strings.Join(lines, "\n")
```

Where `emptyPad = "│" + strings.Repeat(" ", boxW-2) + "│"`

```go
func TestDecisionDialog_ModalStackIntegration(t *testing.T) {
    m := NewRootModel()
    m.Update(tea.WindowSizeMsg{Width: 80, Height: 24})

    // Push DecisionDialog via its Cmd — uses Acknowledge (ℹ Cofre criado)
    cmd := Acknowledge(SeverityInformative, "Cofre criado", "O cofre foi criado com sucesso em ~/documentos/pessoal.abditum.", nil)
    msg := cmd()
    m.Update(msg)

    if len(m.modals) != 1 {
        t.Fatalf("expected 1 modal after push, got %d", len(m.modals))
    }

    // Verify View() doesn't panic with modal on stack
    _ = m.View()

    // Send Enter — should trigger popModalMsg from DecisionDialog
    m.Update(tea.WindowSizeMsg{Width: 80, Height: 24}) // ensure sized
    m.Update(tea.KeyPressMsg{Code: tea.KeyEnter})

    if len(m.modals) != 0 {
        t.Errorf("expected modal stack empty after Enter, got %d modal(s)", len(m.modals))
    }
}
```
