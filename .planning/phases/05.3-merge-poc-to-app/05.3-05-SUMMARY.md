# 05.3-05: Help modal DS-compliant anatomy

## Objective
Fix help modal to match DS dialog anatomy: title in top border, "Esc Fechar" in bottom border, 2-col horizontal padding, 1-line vertical padding, dynamic sizing, scroll indicators.

## What was done
- Rewrote `helpModal.View()` with full DS-compliant anatomy
- Title "Ajuda — Atalhos e Ações" embedded in top border (`╭── Title ──╮`)
- "Esc Fechar" in bottom border with `border.default` color (DS §454)
- 2-col horizontal padding (DS §240), 1-line vertical padding top/bottom (DS §241)
- Dynamic sizing: width = min(70, 80% terminal), height = 80% terminal
- Scroll indicators: ↑/↓ arrows, ■ thumb for position (DS §255-288)
- Scroll navigation: ↑↓ (line), PgUp/PgDn (page), Home/End (top/bottom)
- Added `SetSize(w, h int)` to `modalView` interface in flows.go
- Added `SetSize` stubs to all modal types (modalModel, passwordEntryModal, passwordCreateModal, filePickerModal, stubModal)
- `rootModel.View()` calls `top.SetSize(m.width, m.height)` before rendering

## Key files
- `internal/tui/help.go` — full rewrite
- `internal/tui/flows.go` — modalView interface updated
- `internal/tui/root.go` — View() calls SetSize on modals
- `internal/tui/modal.go` — SetSize stub
