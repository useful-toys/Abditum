# Phase 05.3: merge-poc-to-app - Context (Updated)

**Gathered:** 2026-04-04
**Status:** Bugs found post-execution — needs gap closure

<domain>
## Post-Execution Bugs Found

### Bug 1: Command bar — F1 Ajuda não fixado à direita
- `RenderCommandBar()` monta todas as ações e só garante gap=1, mas o corpo transborda e empurra F1 para fora
- Spec exige: descartar ações de menor prioridade até que F1 caiba (tui-specification-novo.md §940-946)

### Bug 2: Help modal — não segue anatomia do DS
- Título "Keyboard Shortcuts" em inglês, não embutido na borda superior
- Ação "Close" em vez de "Esc Fechar" na borda inferior
- Largura fixa 60 colunas (spec: min(70, 80% do terminal))
- Sem scroll (spec: indicadores ↑/↓/■, navegação PgUp/PgDn/Home/End)
- Tokens de cor hardcoded em vez de tokens do DS (ColorHelpTitle/Key/Sep/Group)
- Spec: tui-specification-novo.md §604-710, tui-design-system-novo.md §436-458

### Bug 3: Barra de mensagens na linha errada
- renderFrame() coloca msgBar entre header e workArea (linha 2)
- Spec: msgBar deve ser penúltima linha, acima da command bar (DS §214-221)
- Ordem correta: header → workArea → msgBar → cmdBar

</domain>

<decisions>
## Gap Closure Decisions

### Bug 1: Command bar truncation
- **D-01:** Truncar ações do corpo (menor prioridade primeiro) até que F1 + gap mínimo caibam
- **D-02:** Se nem F1 sozinho cabe, mostrar apenas "  F1 Ajuda"

### Bug 2: Help modal rewrite
- **D-03:** Título "Ajuda — Atalhos e Ações" embutido na borda superior (`╭── Título ──╮`)
- **D-04:** "Esc Fechar" na borda inferior (`╰── ───── Esc Fechar ──╯`)
- **D-05:** Largura dinâmica: min(70, 80% do terminal)
- **D-06:** Scroll com indicadores ↑/↓/■ na borda direita
- **D-07:** Navegação: ↑↓ (linha), PgUp/PgDn (página), Home/End (topo/fim)
- **D-08:** Tokens do DS: border.default, text.primary (título), accent.primary (teclas), text.secondary (grupos/scroll)

### Bug 3: Message bar position
- **D-09:** Reordenar renderFrame(): header → workArea → msgBar → cmdBar
- **D-10:** Header 2 linhas (app name + separador), workArea = restante - msgBarH - cmdBarH

</decisions>
