## Decisions Made

- Simple ctrl detection: `strings.HasPrefix(raw, "ctrl+")` + `len(rest) == 1 && rest[0] >= 'a' && rest[0] <= 'z'` — more robust than length check since it validates the character is actually a letter
- Function key detection: `raw[0] == 'f'` + all-digit suffix — avoids ambiguity with `fmt`, `false`, etc. if such strings appeared as keys
- `shift+fN` handled via `"Shift+" + formatKeyForHelp(rest)` — recursive composition ensures consistent fN formatting
