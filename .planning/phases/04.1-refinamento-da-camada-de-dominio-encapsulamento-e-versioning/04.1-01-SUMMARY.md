## Summary

Added six private entity methods and fixed two factory initial states in `internal/vault/entities.go`. These methods encapsulate state management and data operations that were previously scattered as direct field access in the Manager.

## What Was Built

- `Cofre.marcarModificado()` — sets `modificado=true` and updates `dataUltimaModificacao` timestamp
- `Segredo.marcarModificacao()` — updates `dataUltimaModificacao` (NOT called for favoriting per D-08)
- `Segredo.zerarValoresSensiveis()` — wipes sensitive campos + observacao via `crypto.Wipe`, sets to nil
- `CampoSegredo.copiar()` — value receiver deep copy with independent `[]byte` slice
- `Segredo.copiar()` — pointer receiver deep copy with `pasta=nil` (caller sets back-ref)
- `Pasta.copiarProfundo(filtrarExcluidos bool)` — recursive deep copy re-establishing all pai/pasta back-references
- Fixed `criarSegredo` factory: `estadoSessao` → `EstadoIncluido` (was `EstadoOriginal`)
- Fixed `duplicarSegredo` factory: `estadoSessao` → `EstadoIncluido` (was `EstadoOriginal`)
