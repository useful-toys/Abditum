    **Step 1: Update `internal/vault/repository.go`:**

    The current interface needs to support FileMetadata for external change detection. However, vault package cannot import storage package (cycle). Solution: define a minimal metadata type in vault package, or keep the interface simple and handle metadata at a higher level.

    **Simplest approach**: Keep `RepositorioCofre` interface minimal. The Manager holds the repository but the TUI/app layer handles external change detection directly via storage functions. The interface stays as:

    ```go
    type RepositorioCofre interface {
        Salvar(cofre *Cofre) error
        Carregar() (*Cofre, error)
    }
    ```

    BUT Manager.Salvar() uses `prepararSnapshot()` then `repositorio.Salvar(snapshot)` — the snapshot is a filtered deep copy. The repository needs the password and salt to encrypt. Options:
    - Repository holds password/salt as state (set during construction or after open)
    - Repository's Salvar receives additional context

    **Decision**: `FileRepository` holds `password`, `salt`, and `path` as internal state. `Salvar` and `Carregar` use these. This keeps the `RepositorioCofre` interface clean.

    Add a comment to repository.go noting that FileMetadata for change detection is accessed via `storage.ComputeFileMetadata` and `storage.DetectExternalChange` directly, not through this interface.

    **Step 2: Create `internal/storage/repository.go`:**

    ```go
    // FileRepository implements vault.RepositorioCofre using the .abditum file format.
    type FileRepository struct {
        path     string
        password []byte
        salt     []byte
        isNew    bool // true if vault has never been saved (use SaveNew on first save)
        metadata FileMetadata // last known file metadata for change detection
    }

    // NewFileRepository creates a repository for an existing vault file.
    // Call after Load() succeeds — password and salt are from the loaded file.
    func NewFileRepository(path string, password, salt []byte, metadata FileMetadata) *FileRepository

    // NewFileRepositoryForCreate creates a repository for a new vault (not yet on disk).
    // First Salvar() call will use SaveNew (direct write, no .tmp).
    func NewFileRepositoryForCreate(path string, password []byte) *FileRepository

    // Salvar implements vault.RepositorioCofre.
    // First save on a new vault uses SaveNew (ATOMIC-03).
    // Subsequent saves use Save with atomic .tmp protocol (ATOMIC-01, ATOMIC-02).
    func (r *FileRepository) Salvar(cofre *vault.Cofre) error {
        if r.isNew {
            err := SaveNew(r.path, cofre, r.password)
            if err != nil { return err }
            r.isNew = false
            // Compute and store metadata + salt from written file
            meta, err := ComputeFileMetadata(r.path)
            if err != nil { return err }
            r.metadata = meta
            // Read salt from file header for future saves
            r.salt, err = readSaltFromFile(r.path)
            if err != nil { return err }
            return nil
        }
        err := Save(r.path, cofre, r.password, r.salt)
        if err != nil { return err }
        meta, err := ComputeFileMetadata(r.path)
        if err != nil { return err }
        r.metadata = meta
        return nil
    }

    // Carregar implements vault.RepositorioCofre.
    func (r *FileRepository) Carregar() (*vault.Cofre, error) {
        cofre, meta, err := Load(r.path, r.password)
        if err != nil { return nil, err }
        r.metadata = meta
        // Read salt from loaded file for future saves
        r.salt, err = readSaltFromFile(r.path)
        if err != nil { return nil, err }
        return cofre, nil
    }

    // Metadata returns the last known file metadata for change detection.
    func (r *FileRepository) Metadata() FileMetadata { return r.metadata }

    // Path returns the vault file path.
    func (r *FileRepository) Path() string { return r.path }

    // UpdatePassword updates the stored password (for master password change).
    func (r *FileRepository) UpdatePassword(password []byte) { r.password = password }
    ```

    Add helper `readSaltFromFile(path string) ([]byte, error)` that reads bytes [SaltOffset:SaltOffset+SaltSize] from the file header.
