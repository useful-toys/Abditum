    RecoverOrphans:
    - If .tmp exists, deletes it (stale from failed save)
    - If vault file is unreadable AND .bak exists, logs warning (does NOT auto-restore — too risky)
    - If neither .tmp nor problems exist, returns nil (no-op)
    - Returns nil (no error) even when .tmp was cleaned up (successful recovery)
    - Does not log file paths or secret data (SEC-PRIV-01)

    DetectExternalChange:
    - Returns (false, nil) when file matches stored metadata (same size + same hash)
    - Returns (true, nil) when file size differs from metadata
    - Returns (true, nil) when file size matches but content (hash) differs
    - Returns error when file cannot be read (permission error, not found)
  </behavior>
  <action>
    **`internal/storage/recover.go`:**

    `func RecoverOrphans(vaultPath string) error`:
    1. `tmpPath := vaultPath + ".tmp"`
    2. If `tmpPath` exists (`os.Stat` succeeds) → `os.Remove(tmpPath)`. If remove fails, return error. If succeeds, continue.
    3. Return nil.

    Note: The ROADMAP mentions restoring `.bak2` → `.bak` when target is unreadable, but this is risky automatic behavior. Keep it simple: just clean stale `.tmp`. The backup files are there for the user/support to manually recover if needed. RecoverOrphans is called before Load — if the main file is corrupt, Load will fail with appropriate error and user can manually rename `.bak`.

    **`internal/storage/detect.go`:**

    `func DetectExternalChange(vaultPath string, metadata FileMetadata) (bool, error)`:
    1. `info, err := os.Stat(vaultPath)` — if error, return `(false, err)`
    2. Fast-path: if `info.Size() != metadata.Size` → return `(true, nil)` — size differs, definitely changed
    3. Slow-path: `data, err := os.ReadFile(vaultPath)` — if error, return `(false, err)`
    4. `hash := sha256.Sum256(data)`
    5. If `hash != metadata.Hash` → return `(true, nil)`
    6. Return `(false, nil)` — file unchanged

    `func ComputeFileMetadata(vaultPath string) (FileMetadata, error)`:
    1. `data, err := os.ReadFile(vaultPath)` — if error, return zero, err
    2. Return `FileMetadata{Size: int64(len(data)), Hash: sha256.Sum256(data)}`, nil

    ```go
    // MigrationFunc transforms JSON payload from version N to version N+1.
    type MigrationFunc func(data []byte) ([]byte, error)

    // migrations maps source version to migration function.
    // Each function transforms payload from version N to N+1.
    var migrations = map[uint8]MigrationFunc{
        // Version 1 is the initial format — no migration needed.
        // Future migrations added here:
        // 1: migrateV1toV2,
    }

    // Migrate applies the chain of migrations from fromVersion to toVersion.
    // Returns the transformed JSON payload.
    // If fromVersion == toVersion, returns data unchanged.
    // If fromVersion > toVersion, returns error (downgrade not supported).
    func Migrate(data []byte, fromVersion, toVersion uint8) ([]byte, error) {
        if fromVersion > toVersion {
            return nil, fmt.Errorf("cannot downgrade from version %d to %d", fromVersion, toVersion)
        }
        if fromVersion == toVersion {
            return data, nil
        }
        current := data
        for v := fromVersion; v < toVersion; v++ {
            fn, ok := migrations[v]
            if !ok {
                return nil, fmt.Errorf("no migration path from version %d to %d", v, v+1)
            }
            var err error
            current, err = fn(current)
            if err != nil {
                return nil, fmt.Errorf("migration v%d→v%d failed: %w", v, v+1, err)
            }
        }
        return current, nil
    }
    ```
