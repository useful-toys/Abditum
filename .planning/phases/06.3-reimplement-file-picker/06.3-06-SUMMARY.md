# Wave 6 Summary — Layout Gap Fixes

## What Was Done

Corrigidos 3 gaps visuais identificados nos golden files do `filePickerModal`.

### Gap 1: renderCaminhoHeader — sem rótulo, com padding

**Antes:** `│Caminho: /golden/stub/path                    │`
**Depois:** `│ /golden/stub/path                            │`

Removido o rótulo "Caminho: ". Adicionado 1 espaço de padding em cada lado.
O path disponível agora usa `innerW - 2` colunas.

### Gap 2: renderFieldRow — rótulo sucinto "Arquivo: " com padding

**Antes:** `│Nome do arquivo  meu-cofre▌                   │`
**Depois:** `│ Arquivo: meu-cofre▌                          │`

Rótulo encurtado de "Nome do arquivo " (16 chars) para "Arquivo: " (9 chars).
Adicionado 1 espaço de padding à esquerda. Campo de input fica mais largo.
Removido o padding interno duplicado (`" "+fieldContent+" "`) que causava largura errada.

### Gap 3: renderFieldSeparator — bug rightLen com -1 extra

**Antes:** `rightLen = innerW - junctionCol - 1` onde `junctionCol = treeW + 1`
→ `rightLen = innerW - treeW - 2` (1 coluna a menos)

**Depois:** `rightLen = innerW - treeW - 1`
→ `1(├) + treeW(─) + 1(┴) + rightLen(─) + 1(┤) = innerW + 2 = modalW ✓`
