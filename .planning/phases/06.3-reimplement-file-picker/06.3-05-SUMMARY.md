### Bug Fix Applied During This Wave
`newGoldenFPK` overrides `currentPath` with `/golden/stub/path` after calling
`loadFilesForCursor()`, so golden files don't embed randomly-named temp directory
paths. Without this fix, every test run generated different paths → golden mismatch.

### Artifacts
- 16 golden files created under `internal/tui/testdata/golden/filepicker-*.golden`
  (8 txt + 8 json)
- All generated via `-update` on first run, verified stable on second run

## Decisions

- **Fixed stub path** (`/golden/stub/path`) instead of real temp dir to make goldens
  portable and stable across runs
- **No scroll indicators in scroll goldens**: with 24-row terminal, `visibleH=15`.
  9 tree nodes and 10 files both fit within 15 rows → no overflow → no `↑`/`↓`
  indicators. Scroll indicator rendering is already covered by `TestFilePickerUpdateBehavior`.
