⚠ IMPORTANT: `newHelpModal` is unexported. The PoC must trigger the help modal via a pushModalMsg, but the PoC is in a separate package and cannot directly call newHelpModal. Instead, the F1 handler in the PoC should push a simple pop-only overlay message or use a local minimal help display. See D-06: "F1 — Handler: pushModalMsg{newHelpModal(actions)}" — but since pocModel is in a different package, Claude should instead export a `NewHelpModal(actions *ActionManager) tea.Model` helper from the tui package, OR implement a simple inline help view in the PoC. Use whichever approach is cleaner without adding complexity.

Simplest clean approach: Add `NewHelpModal(actions *ActionManager) modalView` export to help.go (one line: `func NewHelpModal(a *ActionManager) modalView { return newHelpModal(a) }`). Then the PoC's F1 handler pushes it.

But wait — `modalView` is also unexported (interface). The PoC can store modals as `tea.Model` if we export a `Modal` interface or just store `any` with a type assert. Cleanest: export `NewHelpModal` returning `tea.Model` (since helpModal implements Update+View). The PoC's modal handling stores `tea.Model` and calls `.Update()`.

Actually the simplest approach for a PoC: F1 in the pocModel just toggles `showHelp bool` and renders a simple inline help text in View(). This requires zero exports. The PoC doesn't need the full helpModal machinery. This is the recommended approach — it's a PoC, not production code. Claude decides per "Claude's Discretion" in CONTEXT.md.

**Update() — handles WindowSizeMsg, TickMsg, KeyPressMsg:**
```go
func (m *pocMessagesModel) Update(msg tea.Msg) (tea.Model, tea.Cmd) {
    switch msg := msg.(type) {
    case tea.WindowSizeMsg:
        m.width = msg.Width
        m.height = msg.Height
        return m, nil

    case tui.TickMsg:
        m.msgs.Tick()
        return m, tea.Tick(time.Second, func(t time.Time) tea.Msg { return tui.TickMsg(t) })

    case tea.KeyPressMsg:
        m.msgs.HandleInput()
        if cmd := m.actions.Dispatch(msg.String(), m.showHelp); cmd != nil {
            return m, cmd
        }
        // ESC closes help
        if m.showHelp && msg.String() == "esc" {
            m.showHelp = false
        }
        return m, nil
    }
    return m, nil
}
```

func (m *pocMessagesModel) View() tea.View {
    if m.width == 0 || m.height == 0 {
        return tea.NewView("Initializing...")
    }

    headerStyle   := lipgloss.NewStyle().Width(m.width).Bold(true).Foreground(lipgloss.Color("#a9b1d6"))
    workAreaStyle := lipgloss.NewStyle().Width(m.width)

    const headerH  = 1
    const msgBarH  = 1
    const cmdBarH  = 1
    workH := m.height - headerH - msgBarH - cmdBarH
    if workH < 0 {
        workH = 0
    }

    // Header: plain app name (no vault, no tabs — PoC only)
    header := headerStyle.Render("  Abditum")

    // Work area: help overlay or centered label
    var workContent string
    if m.showHelp {
        workContent = m.renderHelp()
    } else {
        workContent = lipgloss.Place(m.width, workH, lipgloss.Center, lipgloss.Center,
            lipgloss.NewStyle().Foreground(lipgloss.Color("#565f89")).Render("Teste das mensagens"))
    }
    workArea := workAreaStyle.Height(workH).Render(workContent)

    // Message bar (from Plan 01)
    msgBar := tui.RenderMessageBar(m.msgs.Current(), m.width)

    // Command bar (from Plan 02)
    var cmdBarContent string
    if m.showHelp {
        cmdBarContent = lipgloss.NewStyle().Foreground(lipgloss.Color("#565f89")).Render("  esc  Fechar ajuda")
    } else {
        cmdBarContent = m.actions.RenderCommandBar(m.width)
    }

    content := strings.Join([]string{header, workArea, msgBar, cmdBarContent}, "\n")
    v := tea.NewView(content)
    v.AltScreen = true
    return v
}
```

func (m *pocMessagesModel) renderHelp() string {
    titleStyle := lipgloss.NewStyle().Bold(true).Foreground(lipgloss.Color("#7aa2f7"))
    keyStyle   := lipgloss.NewStyle().Foreground(lipgloss.Color("#7aa2f7")).Bold(true)
    sepStyle   := lipgloss.NewStyle().Foreground(lipgloss.Color("#565f89"))
    dimStyle   := lipgloss.NewStyle().Foreground(lipgloss.Color("#565f89"))

    var b strings.Builder
    b.WriteString("\n")
    b.WriteString("  " + titleStyle.Render("Atalhos — Mensagens") + "\n")
    b.WriteString("  " + sepStyle.Render(strings.Repeat("─", 46)) + "\n\n")

    for _, act := range m.actions.All() {
        if len(act.Keys) == 0 {
            continue
        }
        hideTag := ""
        if act.HideFromBar {
            hideTag = dimStyle.Render(" (oculto da barra)")
        }
        b.WriteString(fmt.Sprintf("  %-14s  %s%s\n",
            keyStyle.Render(act.Keys[0]),
            act.Description,
            hideTag,
        ))
    }
    b.WriteString("\n  " + dimStyle.Render("F1 ou Esc para fechar") + "\n")
    return b.String()
}
```
