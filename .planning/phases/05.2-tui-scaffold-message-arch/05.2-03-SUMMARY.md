# Plan 05.2-03: PoC Binary — COMPLETE

## What Was Built

Created `cmd/poc-mensagens/` — a standalone proof-of-concept binary demonstrating the full message bar + command bar system from D-01 through D-06.

## Key Changes

### cmd/poc-mensagens/main.go
- Minimal entry point: `tea.NewProgram(newPocModel()).Run()`
- No vault dependency, no alt-screen option needed (set via `v.AltScreen = true` in View)

### cmd/poc-mensagens/poc_model.go
- `pocMessagesModel` struct: width, height, ActionManager, MessageManager, showHelp flag
- 15 actions registered in Group "Mensagens":
  - F2–F9: 8 visible messages (HideFromBar: false), sorted from Priority 90 to 20
  - shift+F2–F6: 5 TTL variants (HideFromBar: true) — visible in Help only
  - ctrl+Q: exit (Priority 10)
  - F1: toggle inline help (Priority 0, ScopeGlobal)
- `Init()`: starts `tui.TickMsg` tick loop (every second)
- `Update()`: handles WindowSizeMsg, tui.TickMsg, tea.KeyPressMsg; Dispatch + msgs.HandleInput
- `View()`: 4-zone layout — header + work area + message bar + command bar
  - Message bar: `tui.RenderMessageBar()` from Plan 01
  - Command bar: `m.actions.RenderCommandBar()` from Plan 02 (F1 right-anchored)
  - F1 toggles `showHelp` for inline help view
- `renderHelp()`: inline help listing all actions with HideFromBar tags
