# Plan 05.2.1-01: Help Modal DS Dialog Rewrite

## Objective
Rewrite helpModal from a 60-column inline box into a full DS-spec dialog with Portuguese title, action bar, scroll support, and DS token colors.

## What Was Built
- **Full DS dialog**: Removed hardcoded `boxW := 60` and fixed width — dialog now sizes to content, centered by rootModel's `lipgloss.Place`
- **Portuguese title**: "Ajuda — Atalhos e Ações" (replaces "Keyboard Shortcuts")
- **DS token colors**: All colors migrated from hardcoded terminal colors to DS tokens:
  - Border: `#414868` (was `62`)
  - Title/group: `#a9b1d6` bold (was `62`/`14`)
  - Keys: `#7aa2f7` (was `11`)
  - Descriptions: `#a9b1d6` (was `240`)
  - Scroll indicators: `#565f89`
- **Scroll support**: `scrollOffset` and `maxScroll` fields added to struct; navigation via ↑↓/j/k (line), PgUp/PgDn (10 lines), Home/End (top/bottom)
- **Scroll indicators**: ↑/↓ arrows and ■ thumb rendered on right side when content exceeds 20 visible lines; thumb position formula: `round(scrollOffset / maxScroll * (visibleLines - 1))`; arrow prevails over thumb at coincident positions
- **Action bar**: `Shortcuts()` returns `{{Key: "Esc", Label: "Fechar"}}` — rendered by rootModel's command bar when modal active; inline "Esc or F1 close this help" text removed
- **Priority sorting**: Actions within each group sorted by Priority descending (D-10)
- **Exported constructor**: `NewHelpModal()` added for external use (PoC binary)

## Files Modified
- `internal/tui/help.go` — Complete rewrite of View(), buildContent(), Update(), Shortcuts(); added scroll fields, exported constructor
