## Changes Made

### root.go (complete rewrite)

**Struct changes (D-11, D-12):**
- Renamed `preVault *preVaultModel` → `welcome *welcomeModel`
- Changed `modals []childModel` → `modals []modalView` (D-02)
- Removed `flows *FlowRegistry` field
- Added `lastCopyAt time.Time` (D-12: clipboard timer)
- Added `lastRevealAt time.Time` (D-12: reveal timer)

**Constructor (newRootModel):**
- Sets `area: workAreaWelcome` (was workAreaPreVault)
- Registers global actions via new owner-based API: `actions.Register(m, Action{Keys:[]string{...}, Scope, Enabled, Handler})`
- ctrl+q: ScopeLocal, calls Confirm() on unsaved changes
- ?: ScopeGlobal, pushes helpModal

**Update() — D-09 dispatch order:**
1. WindowSizeMsg: SetSize on `liveWorkChildren()` only (NOT modals)
2. pushModalMsg: appends to `m.modals []modalView`
3. popModalMsg: pops from modal stack
4. startFlowMsg: clears orphan modals, sets activeFlow, calls activeFlow.Init()
5. endFlowMsg: sets activeFlow = nil
6. modalResult: routes EXCLUSIVELY to activeFlow (D-03), silently dropped if no flow
7. tickMsg: calls messages.Tick(), broadcasts via broadcast(), re-issues tick
8. Domain msgs: broadcast() to all live models
9. KeyPressMsg: calls messages.HandleInput(), then Dispatch(), then topmost modal, then activeFlow, then activeChild

**Removed from root.go:**
- `buildFlowContext()` method
- `dispatchKey()` method (logic merged into Update)
- `liveModels()` method (replaced by `liveWorkChildren()` + modals iterated separately)
- `chainFlowMsg` case
- All `FlowRegistry`, `flowDescriptor`, `FlowContext` references

**New/updated methods:**
- `liveWorkChildren()`: work-area children only (modals excluded), typed-nil safe
- `broadcast()`: iterates liveWorkChildren() + modals separately
- `activeChild()`: uses workAreaWelcome
- `renderShortcuts()`: new helper for modal command bar
- `renderFrame()`: updated for workAreaWelcome, uses messages.Current() *DisplayMessage
- `View()`: lipgloss.Place of topmost modal over frame content
- `enterVault()`: updated `m.welcome = nil` (was m.preVault)
