# Phase 05.1 Plan 01 Summary

## Objective
Rewrote the core interface contracts and service layer of `internal/tui` to align with `tui-elm-architecture.md`.

## Changes Made

### flows.go (complete rewrite)
- Replaced 5-method `childModel` interface with 3-method version (removed `Context()` and `ChildFlows()`)
- Added `Shortcut` struct
- Added `modalView` interface (Update, View, Shortcuts) — separate from childModel, no SetSize
- Added `modalResult` marker interface with 3 concrete types: `passwordEntryResult`, `passwordCreateResult`, `filePickerResult`
- Added `flowHandler` interface with `Init() tea.Cmd` + `Update(tea.Msg) tea.Cmd`
- Added `startFlowMsg`, `endFlowMsg`, `endFlow()`, `pushModalMsg{modal modalView}`, `popModalMsg`
- Removed: `FlowContext`, `flowDescriptor`, `FlowRegistry`, `chainFlowMsg`

### state.go (partial update)
- Renamed `workAreaPreVault` → `workAreaWelcome`
- Removed `pushModalMsg` struct (moved to flows.go)
- Domain messages unchanged, `tickMsg` unchanged

### flow_open_vault.go (rewrite)
- Removed `openVaultDescriptor` type (references removed FlowRegistry/flowDescriptor)
- Added `Init() tea.Cmd` to `openVaultFlow` to satisfy new `flowHandler` interface

### flow_create_vault.go (rewrite)
- Removed `createVaultDescriptor` type
- Added `Init() tea.Cmd` to `createVaultFlow`

### actions.go (complete rewrite)
- New `ActionScope` type (ScopeLocal, ScopeGlobal)
- Replaced `Action{Key, Priority, HideFromBar}` with `Action{Keys []string, Scope, Enabled func()bool, Handler func()tea.Cmd}`
- Replaced `ActionManager.Register(Action)` + `ClearGroup(string)` with owner-based API: `Register(owner any, ...Action)`, `ClearOwned(owner any)`, `SetActiveOwner(owner any)`, `Dispatch(key, inFlowOrModal bool) tea.Cmd`
- `Visible()` returns enabled actions in active-owner-first order

### messages.go (complete rewrite)
- Added `MsgKind` enum (MsgInfo, MsgWarn, MsgError, MsgBusy, MsgHint)
- Added `DisplayMessage` struct (Text, Kind, Frame)
- Changed `Current()` return type: `string` → `*DisplayMessage`
- Replaced `Set(string)` with `Show(kind, text, ttlSeconds, clearOnInput)`
- Added `Tick()` (TTL countdown + MsgBusy animation frame)
- Added `HandleInput()` (clears clearOnInput messages on keypress)

### dialogs.go (rewrite)
- Added `DialogType`, `DialogOption`, `FilePickerMode` types
- Replaced `NewMessage`/`NewConfirm` with `Message(dtype, title, msg)`/`Confirm(dtype, title, msg, onYes, onNo)`
- Added `PasswordEntry`, `PasswordCreate`, `FilePicker` factory functions
- Added stub modal types: `passwordEntryModal`, `passwordCreateModal`, `filePickerModal` (all implement modalView)
- Added `newDialogModal` helper

### modal.go (migration)
- Removed `width, height int` fields
- Removed `SetSize(w, h int)` method
- Removed `popModalMsg` struct (moved to flows.go)
- Removed `Context()` and `ChildFlows()` methods
- Added `Shortcuts() []Shortcut { return nil }` method
- Updated `View()` to return the box directly (no lipgloss.Place)
- Updated compile-time assertion: `childModel` → `modalView`

### help.go (migration)
- Removed `width, height int` fields and `SetSize` method
- Removed `Context()` and `ChildFlows()` methods
- Added `Shortcuts() []Shortcut` returning `[{Key:"esc", Label:"Close"}]`
- Updated `buildContent`: `act.Key` → `act.Keys[0]` with length guard
- Updated `View()`: returns box directly without lipgloss.Place
- Updated compile-time assertion: `childModel` → `modalView`

### welcome.go (new file, replacing prevault.go)
- Renamed `preVaultModel` → `welcomeModel`, `newPreVaultModel` → `newWelcomeModel`
- Removed `Context()` and `ChildFlows()` methods
- Compile-time assertion: `var _ childModel = &welcomeModel{}`
- `renderHints` helper moved here

### prevault.go — DELETED

### vaulttree.go, secretdetail.go, templatelist.go, templatedetail.go, settings.go
- Removed `Context() FlowContext` method from each
- Removed `ChildFlows() []flowDescriptor` method from each
