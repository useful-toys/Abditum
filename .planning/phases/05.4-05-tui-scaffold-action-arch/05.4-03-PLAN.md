        modal := top.View()

    ```

    4. For each modal line, replace the corresponding frame line at the calculated column offset
    5. Use lipgloss.Width() for column calculations (handles ANSI codes)
    6. Return joined lines

    left := borderStyle.Render("│")

    }
    ```
